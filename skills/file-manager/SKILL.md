# 文件管理

## 概述

当用户的任务与文件系统操作相关时，使用此技能来安全地完成文件管理。

## 使用说明

1. 删除文件前先确认路径是否正确
2. 批量操作时先用 `ls` 或 `find` 预览目标
3. 涉及重要文件时优先备份
4. 使用 `mv` 前先确认目标路径存在

## 示例

- 用户说"帮我找所有的日志文件" → 执行 `find . -name "*.log"`
- 用户说"把 tmp 目录清理掉" → 先 `ls tmp/` 确认内容，再执行删除
